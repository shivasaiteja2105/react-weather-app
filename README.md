# React Weather App

A weather application built with React that shows real-time weather data using WeatherAPI.

## Features
- Search weather by city
- Get weather using current location (Geolocation API)
- Loading spinner while fetching data
- Error handling for invalid cities

## Technologies Used
- React
- Axios
- WeatherAPI
- Geolocation API

## Installation

1. Clone the repository
2. Run `npm install`
3. Add your API key in `.env`
4. Run `npm run dev`
